# Frontend - CareerMitra AI

This frontend is a Vite + React app for the AI Employability workflow.

## Features
- Drag/drop resume upload (PDF)
- JD paste and URL-to-JD import
- Live trust checklist for scam signals
- Match/scam score visualization
- Skill-gap and study-plan rendering
- PDF report export

## Commands
```bash
npm install
npm run dev
npm run lint
npm run build
npm run preview
```

## Environment
Set backend URL if needed:
```bash
VITE_API_URL=http://127.0.0.1:8000
```
